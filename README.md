# Music-video-game 🎶🎮

**Music-video-game**은 음악과 게임을 결합한 리듬 게임으로, 플레이어가 다양한 음악 트랙에 맞춰 박자를 맞추며 점수를 획득하는 게임입니다.

## 📌 주요 기능

- 🎼 **리듬에 맞춘 게임플레이**: 음악 트랙에 맞춰 타겟을 타이밍에 맞춰 클릭하세요.
- 🎶 **커스터마이징 가능한 트랙**: 다양한 음악을 추가하고 난이도를 설정할 수 있습니다.
- 🏆 **점수 시스템**: 최고 점수와 정확도를 기록해 개인 기록을 세워보세요.
